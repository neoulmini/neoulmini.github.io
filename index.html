Neoulmini

A universal mini-language for cross-species and cross-temporal communication

What is Neoulmini?

Neoulmini is a minimalist language created by I. Beklemishchev-Petrov in 2025. It uses just 12 symbols to convey fundamental concepts understandable to any intelligent beings — including animals, AI, and hypothetical extraterrestrials.

Based on principles of semiotics, cognitive psychology, and physical constants.

The 12 Symbols

●

Object
Any material item.

→

Action
Movement, influence.

↑

Increase
Growth, amplification.

↓

Decrease
Reduction, weakening.

⊙

Energy
Light, heat, force.

‖

Barrier
Obstacle, separation.

□

Container
Space, vessel.

!

Danger
Threat, warning.

?

Question
Unknown, inquiry.

§

Connection
Relation, bond.

∞

Eternity
Time, cycles.

⁂

System
Complex structure.

Grammar Rules

  • Linearity: Symbol order = event sequence. Example: ● → ● = "Object acts on object".
  • Grouping: Combine symbols for complex meanings. Example: □● = "Object inside container".
  • 4 Interpretation Levels: From concrete (Level 1) to abstract (Level 4).

Tense/Aspect Extensions (Community Interpretations)

While not formalized in the original grammar (Beklemishchev-Petrov, 2025), users have proposed these temporal markers:

  • → = Present action (default)
  • →∞ = Future/cyclic action
  • ↓→ = Decaying action

Examples:
⊙ →∞ ● = "Energy will act on object" (future)
● ↓→ □ = "Object slowly entering container" (progressive decay)

Message Examples

!● → ↓ — "Dangerous object moves downward" (e.g., falling meteor).

□●§● → ∞ — "Connected objects exist eternally" (e.g., ecosystem).

⊙ → ● → ● → ●□§? → ‖●●● → ↑● → ● → ● → ∞ — Describes our solar system and an invitation for contact (see Appendix in original work).
